# 🔷 Password Generator
# 📌 Overview

This is a Python-based GUI application that generates secure and random passwords with customizable options. It helps users create strong passwords for better security.

# 🎮 How it Works
User selects password length
User chooses character types (letters, digits, symbols)
Password is generated randomly
User can copy password to clipboard
# 🛠 Technologies Used
Python
Tkinter (GUI)
Random Module
String Module
Pyperclip
# 🚀 Features
Custom password length
Option to include letters, digits, and symbols
Secure random password generation
Copy to clipboard functionality
User-friendly GUI
# ▶️ How to Run

Install required library:

pip install pyperclip

Run the file:

python password_generator.py
# 🎯 Outcome

Developed a secure password generator with GUI support, enabling users to create strong and customizable passwords easily
